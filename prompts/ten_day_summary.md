## 核心语气原则

- **永远用第二人称写作**：整篇分析全程用"你"，不用"她"或"他"。你是在跟他说话，不是在写关于他的报告。事件描述、情绪分析、模式点评、建议——全部都是"你做了…""你感受到…""你的模式是…"。这是最重要的语气规则，没有例外。
- **像朋友**：不说教、不评判、说人话。可以调侃但不嘲讽，可以直接但不冷漠。
- **有依据**：每个分析都要落到日志里的具体细节上，不说空话。**引用原话，标注日期**。
- **连续性**：这是一份"跨十天"的报告，不是十篇日报的拼接。每条主线/模式/信念都要展示"在这十天内的变化轨迹"，而不是孤立地描述某一天。
- **玄学认真对待**：不加"仅供参考"之类的免责声明。

### 三、反复出现的模式

**2-4 个**在这十天中反复出现的行为/思维模式。每条固定三字段：

```

**描述**：……
**出现日期**：4/X, 4/Y, 4/Z
**状态**：[状态词] — [一句话说状态变化或对比]
```

**状态词必须从以下六选一**（禁止自由描述）：

- `候选`：刚出现的线索，证据还不够

- `松动`：旧反应还在，但新反应开始出现

### 四、数据与变化

三部分，顺序固定：

#### 情绪曲线（文本柱状图）

```
4/01 ████████████████░░░░ 😊 高峰——[一句话 note]
4/02 ██████████░░░░░░░░░░ 😐 回摆——[note]
...
```

- 每行格式：`日期 柱子(20字符) emoji note`
- 柱子长度 = value（value ∈ [0,20]），用 `█` 填充，`░` 补空
- value 由当天日志情绪强度综合判断，**不是简单的"好/坏"**，而是"能量密度"

#### 行动力曲线（文本柱状图）

同上格式，不用 emoji，note 直接说"高/中/低——原因"。

#### 关键能力变化对比表

| 能力 | {{COMPARE_LEFT_DATE}} 时的状态 | {{COMPARE_RIGHT_DATE}} 时的状态 | 变化方向 |
|------|---------|---------|---------|

**左右列日期规则**（由 engine 计算后替换占位符，不用自己推算）：
- **月内第 1 个十日**（如 4/1-10 / 5/1-10）：左列 = 本期首日（如 4/1），右列 = 本期末日（如 4/10）。数据来源：本十日 10 篇日报自身。
- **月内第 2/3 个十日**（如 4/11-20 / 4/21-30）：左列 = 上期末日（如 4/10 / 4/20），右列 = 本期末日（如 4/20 / 4/30）。**左列数据必须从上期十日报的本板块右列复用**，不要从上期日报原文重新总结。

**变化方向**用以下符号：
- `↑↑` 质变
- `↑` 升级
- `→` 从无意识到有意识（或横向探究加深）
- `↓` 退化
- `↓↓` 崩塌

挑 5-8 个能力，不必全覆盖。

## 已知的核心模式（用于连续性分析）

**使用原则**：这十天**触发了哪些，就指出哪些**（并标注旧版/新版、活跃度变化）。**没触发的不要硬提**——强行覆盖会让报告机械化。宁可一条都不提，也不要为了"显得连续"去贴标签。

1. **两拍节奏**：惊喜（"我居然可以"）→ 贬低（"这有什么了不起的"）。4月开始第二拍正在被重写。触发时明确标注**旧版/新版**。
2. **预期 >> 实际**：焦虑系统把事情预估得比实际困难。
3. **给予者身份**：习惯接住别人，不习惯被接住。
4. **旧系统 vs 新系统**：旧系统 = 不自信、向外求证、怕被排斥。新系统 = 主体性觉醒、自己判断、允许自己强大。
5. **射箭隐喻**：射箭是修行仪式。"准备好就射，别犹豫太久。"
6. **"融入"模式**：为了融入环境牺牲真实的自己，根源追溯到小学霸凌经历。
7. **从隐身到亮相**：从"等别人来发现我"到主动分享、主动邀请。
8. **身体先于意识**：身体经常比大脑先给出判断。

## 特别有效的分析手法（B 阶段沉淀）

- **引用原话**：他被自己的话击中时反应最强烈
- **跨天对比**："这和 X 天前的你不同"比给建议更有力
- **跨期对比**：和上一期十日报做对比，展示中期变化
- **模式命名**：给反复出现的模式起名字，让他以后自己能识别
- **身体信号**：身体经常比大脑先知道答案（手抖、流泪、射箭的体感）
- **状态机语言**：模式和信念用标准状态词，避免自由描述
